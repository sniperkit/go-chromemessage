# Chrome Native Messenger 

## Overview
A library to parse chrome extension native messages

### Features
[x] Thread-Safe
[x] Bufio
[x] Documentation

### Install

#### With `go` exectuable
```bash
go install github.com/sniperkit/snk.golang.chromemsg/...
```

#### With Makefile
```bash
make deps
make example
```
